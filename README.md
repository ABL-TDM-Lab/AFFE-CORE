# AFFE-CORE
Core modules for data processing in PKU-AFFE program

### Git Hooks
1. pre-commit hooks:
   - install pre-commit: `pip install pre-commit`
   - run install: `pre-commit install`
   - run hooks for the first time: `pre-commit run --all-files`
    
2. build `.exe`:
   - `pyinstaller hello.spec`
